This is a project to develop a custom Python implementation of the game "2048" originally created in JavaScript by Gabrielle Cirulli in 2014.

Currently, the core functionality of the game is implemented, but the AutoPlay feature is not yet complete.

The game was built to target:
Python 3.9

and requires modules:
PyQt6
numpy

The game itself can be run in an environment containing the above with the command:

> python gameUIpyqt.py
